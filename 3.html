JavaScript

자바 스크립트 ( / dʒ ɑː V ə ˌ S K R ɪ P t / ) 종종 약칭 JS , 인 프로그래밍 언어 것을 따를 대한 ECMAScript 사양. 자바 스크립트는 고레벨 종종 적시 컴파일 및 멀티 패러다임 . 이 갖는 곱슬 브래킷 구문 , 동적 타이핑 , 프로토 타입 기반 객체 지향 및 일류 기능. HTML , CSS 와 함께 JavaScript는 World Wide Web 의 핵심 기술 중 하나입니다 .웹 사이트의 97% 이상 이 웹 페이지 동작을 위해 클라이언트 측 을 사용 하며 종종 타사 라이브러리를 통합 합니다. 대부분의 웹 브라우저 에는 사용자 장치 에서 코드를 실행 하는 전용 JavaScript 엔진 이 있습니다. 다중 패러다임 언어인 JavaScript는 이벤트 기반 , 기능 및 명령형 프로그래밍 스타일을 지원 합니다. 여기에는 텍스트, 날짜, 정규식 , 표준 데이터 구조 및 DOM( 문서 개체 모델 ) 작업을 위한 API( 응용 프로그래밍 인터페이스 )가 있습니다. ECMA 스크립트 표준은 포함하지 않는 입력 / 출력 등의 (I / O), 네트워킹 , 저장 또는 그래픽 설비. 실제로 웹 브라우저나 기타 런타임 시스템 은 I/O용 JavaScript API를 제공합니다. JavaScript 엔진은 원래 웹 브라우저에서만 사용되었지만 이제는 다른 소프트웨어 시스템, 특히 서버 및 다양한 응용 프로그램의 핵심 구성 요소입니다 . 언어 이름, 구문 및 각각의 표준 라이브러리를 포함하여 JavaScript와 Java 사이에는 유사점이 있지만 두 언어는 서로 다르며 디자인이 크게 다릅니다.
