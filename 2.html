CSS

캐스 케이 딩 스타일 시트 ( CSS은 )는 인 스타일 시트 언어 묘화에 사용되는 프리젠 테이션 에 작성하는 문서의 마크 업 언어를 같은 HTML .CSS는 HTML 및 JavaScript 와 함께 World Wide Web 의 초석 기술입니다 . CSS는 레이아웃 , 색상 및 글꼴을 포함하여 프레젠테이션과 콘텐츠를 분리할 수 있도록 설계되었습니다 . 이러한 분리는 콘텐츠 접근성을 개선 하고, 프레젠테이션 특성의 사양에 더 많은 유연성과 제어를 제공하고, 다음 과 같이 구조적 콘텐츠의 복잡성과 반복을 줄이는 별도의 .css 파일에 관련 CSS를 지정하여 여러 웹 페이지 가 서식을 공유할 수 있도록 합니다. 또한 .css 파일을 캐시 하여 파일을 공유하는 페이지와 해당 형식 간의 페이지 로드 속도를 개선 할 수 있습니다 . 서식과 콘텐츠를 분리하면 화면상, 인쇄본, 음성(음성 기반 브라우저 또는 화면 판독기 사용 ) 및 점자 기반 과 같은 다양한 렌더링 방법에 대해 동일한 마크업 페이지를 서로 다른 스타일로 표시할 수 있습니다. 촉각 장치. CSS에는 모바일 장치 에서 콘텐츠에 액세스하는 경우 대체 형식 지정에 대한 규칙도 있습니다 . 계단식 이름 은 지정된 우선 순위 체계에서 유래하여 둘 이상의 규칙이 특정 요소와 일치하는 경우 적용할 스타일 규칙을 결정합니다. 이 계단식 우선 순위 체계는 예측 가능합니다.
